Electron 多身份浏览器

一个窗口，管理多个身份工作流

Doppel 帮你把账号隔离、标签管理和收藏整理整合到同一套体验里。 适合运营、客服、内容创作者和多账号协作场景。

macOS 直接下载 (.dmg) Windows 直接下载 (.exe)

核心功能

开箱即用，面向日常多账号工作

多身份隔离

每个身份使用独立 partition，会话互不干扰，适合多账号并行操作。

Tab 管理

支持新建、关闭、切换与拖拽排序，重启后可恢复标签顺序。

收藏夹主页

新建标签默认打开 doppel://bookmarks，集中管理常用站点。

身份配置

支持头像、颜色和浏览器画像参数配置，便于快速区分与调试。

4 步开始使用

  1. Step 1

    下载并安装应用

    从 Release 页面下载对应系统安装包（macOS / Windows）。

  2. Step 2

    创建身份并配置外观

    设置名字、头像、颜色，快速区分不同账号工作区。

  3. Step 3

    在同一窗口并行处理多个站点

    通过 Tab 进行切换和排序，保持上下文清晰稳定。

  4. Step 4

    把常用站点加入收藏夹

    地址栏一键收藏，重复 URL 自动去重，减少重复输入。

常见问题

安装时提示“未知开发者”怎么办？

当前发布包可能未签名，系统安全提示属于正常现象。按系统提示允许后即可继续安装。

身份数据是共享还是隔离？

身份会话数据默认隔离，收藏夹为全局共享列表，方便跨身份复用常用网址。

支持哪些系统？

当前发布流程支持 macOS 与 Windows。

准备好开始多身份工作流了吗？

下载 Doppel，减少账号切换成本，保持每个身份的独立上下文和稳定节奏。
